## Project Overview : 

This project is an interactive Excel dashboard created to analyze the annual sales performance of a clothes store in 2022. The dashboard helps visualize sales trends, customer demographics, order status, and channel performance using charts, pivot tables, and slicers.

## Objectives :

-Analyze overall sales and order trends
-Compare sales between men and women
-Identify top-performing states
-Track order status and delivery performance
-Analyze customer age groups and gender
-Evaluate sales performance across different channels

## Dashboard Features :

📈 Order vs Sales Analysis
👨‍🦱 Men vs Women Sales Comparison
🏆 Top 5 States by Sales
📦 Order Status Overview
👥 Age vs Gender Analysis
🛒 Sales Channel Distribution
🎛 Interactive Slicers for filtering data

## Tools & Technologies Used :

-Microsoft Excel
-Pivot Tables
-Pivot Charts
-Slicers
-Data Cleaning
-Dashboard Design

## Key Insights :

-Women contributed the highest percentage of sales
-Maharashtra generated the maximum revenue
-Amazon and Myntra were the top sales channels
-Most orders were successfully delivered
-Adult women customers contributed significantly to sales
